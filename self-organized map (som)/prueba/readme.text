El archivo iris.cpp son los datos de entrada
para la red neuronal.

El archivo irisout.cpp con las salidas
esperadas de las neuronas de salida, son 
los datos target.

El archivo OutErrorIris.dat son los datos
de "error de salida" de cada una de las tres 
neuronas de salida. De este se basa para graficar
(con gnuplot) y ver el comportamiento de 
aprendizaje de la red neuronal.
